Getting Started:
~~~~~~~~~~~~~~~~

Install the latest stable version with pip::

   $ pip install multitensors


**Checkout the docs**:

It's best to checkout the `docs`_. There you'll find detailed
documentation of multitensors}'s features and lots of examples of
how to use them.
